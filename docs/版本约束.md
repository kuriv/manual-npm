# 版本约束

|   名称   |         实例          |               描述               |
| :------: | :-------------------: | :------------------------------: |
|  版本号  |        `1.0.2`        |            指定版本号            |
| 版本范围 |     `>=1.0,<2.0`      |           指定版本范围           |
|  通配符  |        `1.0.*`        |           `>=1.0,<1.1`           |
|  波浪号  | `~1.2` <br/> `~1.2.3` | `>=1.2,<2.0` <br/>`>=1.2.3,<1.3` |
|  折音号  | `^1.2` <br/> `^1.2.3` | `>=1.2,<2.0` <br/>`>=1.2.3,<2.0` |
